# RealCrown Token (RC)

Official GitHub repository for **RealCrown Token (RC)** on the Polygon Network.

RealCrown Token powers the **RealCrown Social Platform**, where users earn crown points for activity such as posting, liking, commenting, and blogging. Crown points can be converted into crypto rewards.

## RealCrown Platform

RealCrown is a social media platform where users can:

• Share posts  
• Upload videos  
• Write blogs  
• Comment and interact  
• Follow other users  
• Earn crown points  
• Convert crown points into crypto  

Posts undergo **AI moderation** to ensure safety and authenticity.
